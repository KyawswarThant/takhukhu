# Takhukhu

Rails User and Admin basic authentication with devise

* Ruby version
    ````
    2.7.5
    ````
    
* Rails version
    ````
    6.1.0
    ````

* System dependencies
    ````
    trailblazer
    devise
    reform-rails
    ````

* Database creation
    ````
    rails db:create
    rails db:migrate
    ````

* Run
    ````
    rails s
    ````
